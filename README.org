* tree-sitter-cabal

Cabal grammar for [[https://github.com/tree-sitter/tree-sitter][tree-sitter]].

This is still very much a work in progress.

** Setup

#+begin_src shell
direnv allow
npm install
#+end_src

** Commands

Running the tests:

#+begin_src shell
npm run test
#+end_src

Building:

#+begin_src shell
npm run build
#+end_src

Parse

#+begin_src shell
npm run parse
#+end_src
