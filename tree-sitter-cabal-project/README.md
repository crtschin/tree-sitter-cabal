# tree-sitter-cabal-project

[cabal.project](https://cabal.readthedocs.io/en/stable/cabal-project.html) grammar for [tree-sitter](https://github.com/tree-sitter/tree-sitter).

## Commands

Building:

```sh
tree-sitter generate
tree-sitter build
```

Parse:

```sh
tree-sitter parse cabal.project
```

Running the tests:

```sh
tree-sitter test
```
